# 🔄 Migration Guide: Capacitor → React Native

## Overview

This guide explains the migration from HTML5/CSS3/Vanilla JS with Capacitor to React Native with Expo.

---

## 📊 Comparison

| Feature | Old (Capacitor) | New (React Native) |
|---------|----------------|-------------------|
| **UI Framework** | HTML/CSS | React Native Components |
| **Language** | Vanilla JavaScript | TypeScript |
| **Build Tool** | Vite | Metro Bundler |
| **Native Wrapper** | Capacitor | Expo |
| **State Management** | Global variables | React Hooks |
| **Styling** | CSS files | StyleSheet API |
| **Navigation** | Manual screen switching | React state-based |
| **API Calls** | Fetch with timeout | Axios |
| **Storage** | localStorage | AsyncStorage |
| **Build Process** | Android Studio | EAS Build (cloud) |

---

## 🎯 What Changed

### 1. **File Structure**

**Old:**
```
mobile/
├── index.html          # All UI markup
├── styles.css          # All styles
├── app.js              # All logic
└── android/            # Native wrapper
```

**New:**
```
mobile-rn/
├── App.tsx                    # Main entry
├── src/
│   ├── screens/              # Screen components
│   ├── services/             # API layer
│   ├── types/                # TypeScript types
│   ├── theme/                # Design system
│   └── config/               # Configuration
└── android/                  # Generated by Expo
```

### 2. **UI Components**

**Old (HTML):**
```html
<div class="property-card" onclick="selectProperty('plaza')">
  <div class="property-icon">🏨</div>
  <div class="property-name">Plaza Residency</div>
</div>
```

**New (React Native):**
```tsx
<TouchableOpacity
  style={styles.propertyCard}
  onPress={() => onSelectProperty('plaza')}
>
  <Text style={styles.propertyIcon}>🏨</Text>
  <Text style={styles.propertyName}>Plaza Residency</Text>
</TouchableOpacity>
```

### 3. **Styling**

**Old (CSS):**
```css
.property-card {
  background: white;
  border-radius: 16px;
  padding: 32px;
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.1);
}
```

**New (StyleSheet):**
```tsx
const styles = StyleSheet.create({
  propertyCard: {
    backgroundColor: colors.white,
    borderRadius: 16,
    padding: 32,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 4 },
    shadowOpacity: 0.1,
    shadowRadius: 16,
    elevation: 8, // Android shadow
  },
});
```

### 4. **State Management**

**Old (Global Variables):**
```javascript
let currentProperty = null;
let currentGuests = [];

function selectProperty(property) {
  currentProperty = property;
  showScreen('dashboard-screen');
}
```

**New (React Hooks):**
```tsx
const [selectedProperty, setSelectedProperty] = useState<Property | null>(null);
const [guests, setGuests] = useState<Guest[]>([]);

const handleSelectProperty = (property: Property) => {
  setSelectedProperty(property);
  setCurrentScreen('dashboard');
};
```

### 5. **API Calls**

**Old (Fetch with manual timeout):**
```javascript
async function apiFetch(url, options = {}) {
  const controller = new AbortController();
  const timeoutId = setTimeout(() => controller.abort(), 12000);
  
  try {
    const response = await fetch(url, {
      ...options,
      signal: controller.signal
    });
    clearTimeout(timeoutId);
    return response;
  } catch (error) {
    clearTimeout(timeoutId);
    throw error;
  }
}
```

**New (Axios with built-in timeout):**
```typescript
const api = axios.create({
  baseURL: API_BASE_URL,
  timeout: API_TIMEOUT,
  headers: {
    'Content-Type': 'application/json',
  },
});

export const guestApi = {
  getStats: async (property: Property): Promise<DashboardStats> => {
    const response = await api.get(`/guests/stats?property=${property}`);
    return response.data.data;
  },
};
```

### 6. **Storage**

**Old (localStorage):**
```javascript
localStorage.setItem('staffLoggedIn', 'true');
const loggedIn = localStorage.getItem('staffLoggedIn');
```

**New (AsyncStorage):**
```typescript
await AsyncStorage.setItem('staffLoggedIn', 'true');
const loggedIn = await AsyncStorage.getItem('staffLoggedIn');
```

### 7. **Navigation**

**Old (Manual DOM manipulation):**
```javascript
function showScreen(screenId) {
  document.querySelectorAll('.screen').forEach(screen => {
    screen.classList.remove('active');
  });
  document.getElementById(screenId).classList.add('active');
}
```

**New (React state):**
```tsx
const [currentScreen, setCurrentScreen] = useState<Screen>('login');

const renderScreen = () => {
  switch (currentScreen) {
    case 'login':
      return <LoginScreen onLogin={handleLogin} />;
    case 'dashboard':
      return <DashboardScreen />;
    // ...
  }
};
```

### 8. **Forms**

**Old (HTML forms):**
```html
<input
  type="text"
  id="guest_name"
  placeholder="Enter guest name"
  required
/>
```

**New (Controlled components):**
```tsx
const [guestName, setGuestName] = useState('');

<TextInput
  style={styles.input}
  value={guestName}
  onChangeText={setGuestName}
  placeholder="Enter guest name"
/>
```

## 📦 Build Comparison

### Old (Capacitor)

```bash
# Build web assets
npm run build

# Sync to Android
npx cap sync

# Open Android Studio
npx cap open android

# Build APK in Android Studio (manual)
```

**Time:** ~5-10 minutes (manual steps)  
**Requires:** Android Studio installed

### New (React Native + Expo)

```bash
# Build APK (cloud)
eas build -p android --profile preview
```

**Time:** ~10-15 minutes (automated)  
**Requires:** Nothing! (cloud build)

## 🔄 Migration Steps (If You Want to Migrate More Features)

### Step 1: Install Dependencies
```bash
cd mobile-rn
npm install
```

### Step 2: Update API URL
Edit `src/config/api.ts`:
```typescript
export const API_BASE_URL = 'https://your-api-url.com/api/v1';
```

### Step 3: Test on Expo Go
```bash
npm start
# Scan QR code with Expo Go app
```

### Step 4: Build APK
```bash
eas build -p android --profile preview
```

## 🐛 Common Issues & Solutions

### Issue: "Can't connect to API"
**Solution:** Check API URL in `src/config/api.ts`

### Issue: "Build fails"
**Solution:** 
```bash
rm -rf node_modules
npm install
```

### Issue: "App crashes on startup"
**Solution:** Check console logs with `npm start`

### Issue: "Styles look different"
**Solution:** React Native uses different units (no px, just numbers)
